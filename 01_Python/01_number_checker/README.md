# Number Checker

## Overview

Number Checker is a beginner-friendly Python project designed to teach kids and young learners how to use input, conditions, and comparison operators.

## What the Program Does

The user enters a number, and the program checks whether the number is:

- Positive
- Negative
- Zero
- Even
- Odd

## Learning Objectives

By completing this project, students will learn:

- How to take input from the user
- How to convert input into an integer
- How to use if / elif / else
- How to use comparison operators
- How to use the modulo operator %

## Concepts Covered

- input()
- int()
- if / elif / else
- Comparison operators
- Modulo operator
- Basic problem-solving
