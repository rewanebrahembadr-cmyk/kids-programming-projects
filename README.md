# 01_Python - Python Projects for Kids

This folder contains beginner-friendly Python projects designed to teach kids and young learners programming step by step.

## Learning Goals

Through these projects, students will learn:

- Variables
- Input and output
- Conditions
- Loops
- Functions
- Lists
- Problem-solving
- Simple game logic
- Menu systems
- Basic user interaction

## Projects List

| No. | Project | Concepts |
|---|---|---|
| 01 | [Number Checker](01_number_checker) | Variables, input, conditions, comparison operators |
| 02 | [Simple Calculator](02_simple_calculator) | Arithmetic operators, conditions, user input |
| 03 | [Guess the Number](03_guess_the_number) | Loops, random module, conditions, attempts counter |
| 04 | [Quiz Game](04_quiz_game) | Lists, dictionaries, loops, score system |
| 05 | [To-Do List](05_to_do_list) | Lists, functions, menu system, add/view/delete |

## Teaching Purpose

These projects are designed as practical activities for teaching Python basics to kids and beginners.

Each project focuses on simple logic, clear steps, and interactive learning.

## Folder Structure

Each project folder contains:

- Python source code
- Project README
- Learning objectives
- Teaching notes
- Example output
