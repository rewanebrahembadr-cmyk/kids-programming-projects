# Kids Programming Projects

This repository contains beginner-friendly programming projects designed to teach kids and young learners programming step by step.

The projects are organized by technology, starting with Python. Scratch and MIT App Inventor projects will be added later as part of a complete kids programming teaching portfolio.

## Current Track

### 01_Python

Python projects designed to teach programming basics through simple, interactive, and beginner-friendly activities.

## Future Tracks

- Scratch Projects
- MIT App Inventor Projects
- AI Activities for Kids

## Purpose

The goal of this repository is to build a structured teaching portfolio for kids programming and EdTech learning.

## Target Learners

- Kids
- Young beginners
- Students learning programming for the first time

## Teaching Approach

Each project focuses on one or more programming concepts such as variables, conditions, loops, functions, lists, and problem-solving.
